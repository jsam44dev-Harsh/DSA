# Data Structures and Algorithms in Java

A personal repository for practicing **Data Structures and Algorithms (DSA)** using **Java**.  
This repo contains topic-wise code examples, problem-solving practice, and implementations of commonly used algorithms and data structure concepts.

---

## 📚 Topics Covered

- Arrays
- Bit Manipulation
- HashMap / HashSet
- Linked List
- Recursion
- Strings
- Patterns
- Object-Oriented Programming (OOP)
- Practice Problems
